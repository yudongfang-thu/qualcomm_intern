# Qualcomm Intern Notes

This repository stores portable notes and planning documents for internship work.

## Contents

- `docs/QNN_QAIRT_学习笔记.md`: Chinese learning notes for the Qualcomm QNN / QAIRT SDK documentation.
- `docs/qnn_outline.tsv`: Topic outline extracted from the public QNN documentation tree.
- `docs/pi0_vla_ros2_deployment_plan.md`: Software-first deployment plan for pi0 VLA, QNN, and ROS2 robot integration.
- `docs/phase1_qnn_onnx_resnet_runbook.md`: Copy-paste-oriented Phase 1 runbook for ONNX/ResNet float deployment with QNN.
- `docs/phase1_assets_and_baseline.md`: ResNet model/input collection and ONNX Runtime baseline instructions.
- `docs/qnn_backend_inference_modes.md`: Quick guide to QNN CPU/GPU/HTP/DSP/LPAI/Saver backends and execution modes.
- `scripts/phase1/`: Download, baseline inference, baseline validation, and QNN-vs-baseline comparison scripts for Phase 1 correctness checks.

## Notes

This repository intentionally does not include raw mirrored Qualcomm documentation pages or large scraped corpora. Keep public content limited to original notes, task plans, small indexes, and code that is safe to share.
